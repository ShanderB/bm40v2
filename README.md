# How start

 ####  hex files
>
> - Install qmk tookbox.
> - Active 'reset' mode on your keyboard. 
> - flash the hex file on your keyboard. 


 ####  Build your own files
>
> - Install QMK MSYS.
> - Open the prompt. 
> - Build and flash (as 'useful commands').
